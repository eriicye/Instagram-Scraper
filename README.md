# Instagram Scraper ⛏️
This was a project convinced by my friend.

I decided to test my python and API skills through this project.

It utilises the instagrapi library.

Overall, this was an incredibly fun program to write; it taught me a lot about APIs and JSON

## Features 🎯
- User Input 👤
- Web Scraping 🌐
- Program Output 💻
- User Login 🔑
- Retrieve User Info 🪝

## Installation 🚀

### 1. Dependencies 📋
- Python 3.0+
- IDE or Editor for Python
- Instagrapi library installed

### 2. Clone Project 🖨️
Run the following command:  
```git@github.com:eriicye/Instagram-Scraper.git```  

### 3. Open Project 📂
Open the project in your preferred Editor or IDE
